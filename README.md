# Python Module and Function Import

This Python program demonstrates how to define a module and import a specific function from that module into another program.

## Files Used

- `my_module.py`
- `main_program.py`

## Operations Performed

- Creating a module
- Defining a function inside the module
- Importing a specific function using `from ... import`
- Calling the imported function

## Python Code

The module file contains a function named `greet()`.  
The main program imports only that function and uses it.
